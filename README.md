# 👋 Hi, I'm Reagan Ochola

### 🤖 Bridging Industrial Automation and AI Intelligence

I am an **AI Engineer & Automation Specialist** dedicated to building intelligent systems that solve real-world industrial and agricultural challenges in Africa. From "Factory Brains" that remember every SOP to Computer Vision pipelines that never tire, I build tech that scales.

---

### 🚀 Specialized Domains

#### **🏭 Manufacturing Intelligence & RAG**
I develop **Hybrid RAG (Retrieval-Augmented Generation)** systems that digitize tribal knowledge.
- **Featured**: **ForgeIntelligence** — An internal factory AI that processes SOPs, manuals, and live machine telemetry to reduce downtime and optimize inventory.

#### **👁️ Computer Vision & Quality Control**
End-to-end automated visual inspection systems.
- **Featured**: **Defect Detection Pipeline** — A PyTorch/OpenCV system detecting sub-millisecond defects in packaging lines, reducing manual QC labor by 80%.

#### **📊 Geospatial & Predictive Analytics**
Turning data into actionable insights for the agricultural sector.
- **Featured**: **AgroDashboard** — Spatial rainfall and yield analysis using Python.
- **Featured**: **Maize Yield Insights** — Power BI & SQL dashboards for crop trend analysis in East Africa.
